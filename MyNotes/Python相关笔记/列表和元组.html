列表和元组

列表和元组

一.列表方法（[]表示列表）

1.append 在最后添加元素
list1 = ['a','b']
list1.append('c')
2.count 用来统计某个元素在列表中出现几次
word = ['world','asfas','world','aca','world']
word.conunt('world')
3.extend 把另外一个列表的元素添加到当前列表
list1 = ['a','b']
list2 = ['c','d']
list1.extend(list2)
#以上代码相当于以下代码：
list1+list2
4.index 获取元素在列表中第一个匹配项的下标值
list1 = ['a','b','c','a']
list1.index('a')
#这里会得出0，a在列表中的第一个匹配项的下标是0
5.insert 可以在任意位置添加元素
list1 = ['a','b','c','a']
list1.insert(2,'b')
6.pop 删除元素（默认为最后一个）
list1 = ['a','b','c','a']
list1.pop(0)
7.sort 排序
list1 = ['a','c','b','a']
list1.sort()

# 降序
vowels.sort(reverse=True)
